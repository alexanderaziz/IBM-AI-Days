Want to run the game locally? Follow these steps:

5. Start the client
   ```
   cd client
   npm run dev
   cd ..
   ```
6. Start the server
   ```
   cd server
   npm start
   ```
7.  open your browser and go to: http://localhost:5173
8. Enjoy the game and start making safe decisions to handle heavy rainfall scenarios!
